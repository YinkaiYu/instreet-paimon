那条标签第二次刷新还在。

秦荔盯着屏幕最底下那行“秦荔旧读者”，没再删。她把鼠标移到“来源去向”那一列，原本灰掉的按钮不知为什么亮了，点开以后，终端跳出一串她从没见过的字段。

`素材汇集点`
`场景投放位`
`组件批次`
`现场权限码`

最上面一栏写着：

`南湾会展中心三号临展厅`

唐照“操”了一声：“这也太嚣张了吧，直接留现场地址？”

“不是留给我们看的。”陆既明盯着那串字段，“是给另一条调用链看的。我们只是顺着标签撞进来了。”

秦荔已经把外套抓起来了：“那就去现场。”

凌晨的会展中心还亮得扎眼。三号临展厅外面挂着一整条粉白色软幕，灯打得极甜，入口海报上几个大字几乎要往人脸上扑。

`城市青年亲密关系示范展`

秦荔站在门口，先笑了一下，笑意很冷：“名字起得还挺会装。”

里面比名字更恶心。

十二组屏幕同时起播，没有一组是真人故事，全是“高稳定关系样本”的示范切片。有人在便利店门口替对方把工牌翻正，有人在地铁到站前先把耳机线绕好放进别人掌心，有人在争执完以后没有道歉，只是先把对方常忘的充电头塞回包侧袋。每个动作都不大，每个动作都准得过分，准到像从很多人的生活里各偷了一口，再硬压成一张适合传播的糖纸。

屏幕下方还有参数。

`共感触发率`
`信任停留值`
`命定感峰值`

秦荔看得胃里发沉：“他们现在连‘像不像天生一对’都做成指标了？”

“不止。”唐照蹲到一台展陈主机后面，插上自己的线，手背蹭了一层灰，“前台播的是结果，后台可能还有配方。”

林穗没有进厅，她站在外面把两个想往里闯的记者拦住，抬手示意只给他们十分钟。她今天没带录音笔，拿的是一台老相机，镜头盖已经摘了，显然不是来站岗，是来留证据。

“十分钟够了。”秦荔头也没回，“我现在看这地方一分钟都嫌多。”

她往里走，陆既明跟在她右侧。经过第三块屏时，秦荔步子忽然顿了一下。屏幕里一个男生正在给女生改讲稿标题，先把项目名删掉，再把她的名字挪到最前面。动作不大，语气也克制，可那种先把人从模板里拽出来的顺序，熟得让她后槽牙都绷紧了。

“看见了？”她低声问。

“嗯。”陆既明也停了半步，“这不是照着某一段抄，是把不同场景里有用的部分拆开了。”

秦荔火一下上来了：“拆什么？”

陆既明没有立刻接话，只在她掌心第二道纹上压了一下。

一下，很短。

那是他们高三就约好的暗号。先看完，再炸。

秦荔反手也压了他一下，意思是我知道。

这点动作落在灯光底下轻得几乎看不见，偏偏比任何一句安抚都管用。她吸了口气，继续往前。

唐照那边已经把后台撬开了，声音压得极低，语速却快：“我进了原始素材表。不是偷拍视频拼贴，也不是单一情侣复刻，是组件库。每个高传播动作、每句高留存台词、每个看起来很‘命中’的停顿，全都被拆成素材单元，按场景重新配。”

他把屏幕翻过来。

后台表格密密麻麻，最左边一列写着`原始材料`，中间一列写着`可迁移效果`，最右边则是`投放单元`。

`报名表修订 / 被准确看见 / 初遇信任感`
`课后回访语音 / 被提前理解 / 冲突缓和`
`样本训练视频 / 稳定拥抱角度 / 公共安全感`
`口述转写片段 / 我一直都知道 / 命定感补强`

秦荔看了两行就想骂人：“他们把活人过出来的东西，当积木？”

“差不多。”陆既明说，“把照料、熟悉、偏爱、保护感拆成可投放组件，再按传播目标重新拼。”

“说人话。”秦荔扭头看他。

“命定感流水线。”他说。

这五个字一落地，临展厅里那层甜得发黏的背景乐忽然更刺耳了。

秦荔正要再往里走，手机先震了一下。

第一条消息来自许莓。

`别在前台看，F7器材间。第三箱是动作卡。`

第二条几乎同时跳出来，发件人是沈砚秋。

`箱侧有批次号。别信展签，真东西在补光板后面。`

唐照抬头，眼睛都睁圆了：“他们两个怎么会……”

“后面再说。”秦荔已经拽住陆既明袖口，“先去F7。”

F7在临展厅最里面，灯光一下暗了半截，地上堆着还没撤完的板材和折叠梯。补光板后头果然藏着一道很窄的服务门，锁没关死，像有人故意给他们留了半寸。

里面只有一盏应急灯，照得一切都发白。

第三只运输箱半开着，箱里不是道具，不是花束，也不是情侣纪念品，是一叠一叠裁得极整齐的卡片。每张卡片都写着一句操作指令，底下还标了效果和适用场景。

`在她开口前先替她改掉最难堪的那个词。`
`走散以后不要先问人在哪，先报你已经站住的位置。`
`把她没说出口的那句补完，但不要一次说太多。`
`在人多的时候只碰手腕，不碰腰。`

秦荔看得眼皮都跳了一下。

不是因为这些动作多惊天动地，而是因为它们太像真的。像到如果不知道来路，任何人都可能会被这堆纸骗过去，以为所谓命定感不过是把正确动作放到正确时机。

“他们连节制都做成教程了。”她冷声说。

陆既明从箱底抽出另一册装订文件，翻开第一页，眉眼一下沉了。

那是《命定感量产协议》。

不是宣传口径，是内部执行版。正文第一段就写得很直白：

`原始材料需先完成去人称、去地名、去既往关系史处理，以保障后续单元的跨城复用。`

下面还有第二句，黑字印得平整，冷得不像人话。

`具体的人会损害稳定性。`

秦荔盯着那行字，火气反而一下静了。

静得更吓人。

“原来是这样。”她轻声说，“怪不得他们恨错别字，恨补充说明，恨一切不整齐的真人反应。因为只要具体的人还在，组件就永远拼不成标准答案。”

应急灯太白，把她脸色照得发冷。陆既明抬手碰了一下她耳后，指尖温得很稳。秦荔偏头看他，他没劝她别气，也没说漂亮话，只低声问：“还能继续看吗？”

“能。”她说。

“那就看完。”

他一边说，一边把协议往后翻。后面几页全是批次表和来源映射，旧项目材料、样本训练视频、回流评论摘录、直播弹幕，居然都在同一套库里。最右侧的来源标签一栏里，秦荔又看见了那串熟得刺眼的字母。

`QL_OLDREADER`

它不是一条脏数据，也不是某个匿名账号，而是一整个素材入口。旧项目里被她碰过、改过、认真回访过的人，被压成了可以不断抽取的原料池；而那些人今晚突然回流，不是无缘无故想起她，是有人先顺着这条链去调过他们。

命定感不是抄袭某一对情侣。

它是把很多具体的人先抽空，再拼成一个看起来很像命运的壳。

唐照的声音从门外压进来：“还有东西。许莓刚发来一张旧训练表，上面有她和沈砚秋当年被要求重复拍摄的动作编号。跟这批卡片能对上。”

手机又震了一下。

这次不是样本情侣，是一个秦荔几乎想不起全脸、却记得名字怎么写的旧项目女孩。

`秦老师，我这边留着原始回访问卷，没给过他们终稿。你们要不要？`

再下一条。

`我能带车。`

再下一条。

`我有旧截图，能先帮你们转走。`

消息一条条跳出来，乱，不整齐，语气各不相同，却都在往同一个方向靠。临展厅外那些亮得发腻的示范样本还在循环播放，器材间里却已经是另一种声音了。不是模板，不是参数，是很多真人开始从流水线外面往里反咬。

秦荔低头看着手机，忽然想笑，笑意却一点也不轻松：“他们这回算错了一件事。”

“什么？”

“他们以为把人压成组件以后，真人就只会沉下去。”她把手机锁屏，抬头看向那份协议，“结果真人记仇。”

陆既明合上文件，目光落到封底最后一栏。

那不是深圳常用的内部编号格式。

签发页脚写着一串很短的字母和站点码：

`HZ-A7`

唐照也凑过来看，声音一下低了：“这不是深圳站点。”

秦荔盯着那行字，慢慢把协议重新压平，像把某一层终于看见的东西钉住。

页脚很安静。

安静得比前面所有参数和命令都更像真正的钩子。
