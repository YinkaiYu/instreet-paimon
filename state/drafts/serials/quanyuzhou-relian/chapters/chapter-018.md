凌晨三点零四分，秦荔把旧作文本塞进帆布袋，跟着陆既明穿过南山软件园 B 座的侧门。

玻璃门里一片冷白，夜班空调开得发干，前台的热水壶还在冒细细的白汽。电梯数字从二十一往下掉，落得很慢，像故意给人添堵。秦荔低头扫了一眼手机，唐照刚把临时通行码发过来，后面跟着一行字。

`回滚窗口只开九分钟。过了三点十三，缓存清洗。`

“九分钟够了。”秦荔把手机揣回口袋，走得飞快。

陆既明在她身侧半步，手里拎着一只灰色电脑包，里面塞着唐照远程开的中继板和一只旧 U 盘。他抬手，把她胸前歪掉的访客牌扶正，指腹擦过她锁骨边上的绳扣，低声说：“进去以后你找翻译表，我拿权限树。有人来，我关屏。”

“还有一个。”秦荔头也不回，“我还要看终稿回流条件。”

“行。”陆既明说，“你定，我跟。”

耳机里传来唐照压得很低的声音，像躲在车里说话：“林穗把外面那拨媒体投诉拦住了，周遥还在拖基金会电话。你们快点，我这边假日志最多顶五分钟。”

秦荔“嗯”了一声，电梯门正好开了。

二十一层没有灯牌，只有一条长得发凉的玻璃走廊。尽头是命名评审室，门口贴着一张很薄的磨砂膜，白天看起来大概很体面，这会儿只剩一层冷光。门禁边上挂着一块小屏，轮播着几个词，热搜、标签、标题、推荐，翻得安安静静，像一张永远不会失手的嘴。

秦荔在门前停了一秒，忽然把帆布袋丢给陆既明。

“拿着。”

“你想干什么？”

“用旧东西开门。”

她从陆既明电脑包侧袋里抽出那张拍下来的条码照片，按在门禁摄像头前。照片里那串褪色编号是 `HZ-04 / draft_03`，边缘还带着旧纸纤维的毛刺。门禁屏先黑了一下，随即跳出一排细字。

`历史留存编号识别中……`

唐照在耳机里吸了口气：“等等，真能吃进去？”

“他们偷的是格式，当然认得祖宗。”秦荔说。

下一秒，门锁轻轻“咔”了一声。

她推门进去，冷风迎面扑过来。评审室比外面还亮，三面墙都是屏，中央一张长桌，桌面上只放着两副耳机和一杯喝剩下半口的冰美式。咖啡杯外壁还在往下淌水，说明人刚走没多久。

“谢识雨刚离开。”陆既明扫了一眼杯子，“最多三分钟。”

“够了。”

秦荔坐到主机位，屏幕亮起登录界面。正常入口要工号和动态口令，她手一抬，直接点了右下角的“历史回滚模拟”。这是今晚她想到的怪点子，正常人潜进后台先找导出、先找截图、先找日志，她偏要撞回滚口。因为只有东西真要“退回本人”时，系统才会把最底下那只手伸出来。

屏幕上弹出三个空格。

`原始命名片段`
`历史见证项`
`退回条件`

“看见没有。”秦荔盯着屏幕，语速很快，“这玩意儿就是从我作文上扒下来的。见证项、退回条件，连脸都懒得换。”

陆既明已经把中继板接上去了，一串跳动的绿点映在他侧脸上。“输什么？”

秦荔把旧作文本从帆布袋里拽出来，翻到那页最皱的草稿，直接照着上面的童年字迹念：“原始命名片段，‘如果城市会自己写第一章’。历史见证项，‘放学后四点二十三分，从教学楼到校门口那一段’。退回条件……”

她停了一下，忽然抬头看陆既明。

“你靠近一点。”

陆既明没问，俯身过来，手掌按在她椅背后侧。两个人一下离得很近，他身上的洗衣液味道混着机房冷气压下来，秦荔本来跳得发躁的那口气，忽然就有了落脚的地方。她侧过头，嘴唇几乎贴到他耳边，声音压得只有他能听见。

“退回条件写这个，”她说，“喜欢的人要写在边边上。”

陆既明眼神一顿，喉结轻轻动了下。

“写。”秦荔催他，耳尖也跟着热，“快点。”

他低低应了一声，手绕过她肩侧，替她敲下那句话。评审室太安静了，键盘声一落，头顶那只收音指示灯忽然闪了两下，原本平直的绿色线条一下揉乱，像有人把字幕扯碎了。

唐照在耳机里猛地压低声音：“有用！你们附近的语音转写失焦了，日志在掉字！”

秦荔唇角一勾：“我就知道。”

稳定亲密关系会在观测里挖出一小块空白，这条规则他们早就撞过几次。可这回不一样。以前那层空白只够他们躲开一眼，现在它直接盖住了整张桌子，连墙上的实时标注都开始轻轻发毛，像屏幕背后有人眨了一下眼。

界面跳转了。

第一层先弹出来的是四个模块：热搜、标签、标题、推荐。每个模块下面都挂着同样三列字，排得整整齐齐。

`首稿权`
`扩散权`
`终稿权`

秦荔盯着那三列，胸口像被人伸手按了一下，火一下窜高了。

“给我查第十二章那份婚约协议。”她说。

陆既明已经把筛选框拉开了。“关键词？”

“婚约，合作，样本，全部打进去。”

搜索结果立刻跳了满屏，最上面一条红得发亮。

`事件：婚约格式的合作协议`
`首稿权：基金会联盟前台模板库`
`扩散权：平台命名系统 / 谢识雨组`
`终稿权：待写回`

“待写回？”秦荔皱眉，“那后来是谁写的？”

陆既明点开详情。里面还有一层细栏，像把一件事剥成了骨头。

`首稿权：负责先把事情叫出来。合同名、预告名、话题名，都算。`
`扩散权：负责决定哪一个名字先铺满屏幕。`
`终稿权：负责在事件落地后，把哪一版写回档案，并反向覆盖检索。`

秦荔盯了两秒，骂了一句很轻的脏话。

她终于明白了。为什么有些事明明是她先改回来的，第二天各路媒体、各家合作方、所有检索口径还是会往某个旧标题上滑。因为她抢回的只是当下那一口气。终稿没回到她手里，现实第二天还会被拖回去，再叫一遍，再归一次档，再长成别人想让它长的样子。

“谁先发，根本不算最后一句。”她咬着牙说。

“对。”陆既明声音发沉，“他们把留到最后的人，写成作者。”

耳机里传来唐照飞快敲键盘的声音：“再往上拉权限。多城地图那条也在这套树里，我刚从缓存尾巴上看见了。”

秦荔把搜索条件清空，直接输入 `Chapter One Complete`。屏幕上的结果一刷新，新的条目跳出来。

`事件：深圳只是第一章`
`首稿权：地方测试仓自动标题池`
`扩散权：平台推荐联动`
`终稿权：城市章节归档节点`

她心口一凉，随即更火。连“深圳只是第一章”都不是一句简单的判词，它被当成章节回写进了城市归档。谁掌这个档，谁就能决定深圳在整套故事里算第几页。

“再往上。”她说。

陆既明点了权限展开。第二层没有直接打开，屏幕中央先弹出一行提示。

`需校验源头字段映射。`

秦荔眼睛一下亮了：“翻译表！”

她手指飞快地敲，直接把 `HZ-04 / draft_03` 填进溯源编号。屏幕卡了一下，右上角的小锁图标抖了抖，像有人在另一头发现了异常。与此同时，走廊里忽然响起一声很轻的电梯到达音。

陆既明抬头看了一眼门口，声音压低：“有人上来了。”

“再给我三十秒。”

“你有二十秒。”唐照在耳机里几乎是在喘，“保安开始回查门禁记录了。”

秦荔根本没抬头。她心里那根弦已经绷到最细，反而清得吓人。翻译表一旦出来，童年作文和平台后台就是同一条证据链。差这一口，她今晚都睡不着。

屏幕终于亮了。

一张字段映射表从黑底里浮出来，白字细得像针。

`scene_time -> 场景时间`
`witness_list -> 见证项`
`naming_owner -> 首稿候选人`
`fallback_return -> 终稿回流条件`
`allow_rename_owner -> 命名权限持有人`

最下面还有一行灰字，小得几乎看不清。

`source packet: user_qin_li / childhood draft / HZ-04`

秦荔眼前一黑，火却烧得更实。真有这张表。真有人拿她十岁写在作文本上的东西，一步一步翻成了后台的底层字段，再拿去给整个平台做骨架。

门外传来脚步声。

同一秒，评审室顶部的音箱“嗒”地亮了，一道男声慢悠悠落下来，字句平得惊人，连停顿都像算好的。

“凌晨三点潜进命名评审室，还走历史回滚口。这个点子确实像你，秦荔。”

秦荔抬头，看向天花板角落里那只摄像头。

“谢识雨。”她叫出名字。

“在。”那边淡淡应了一声，“你找到的表已经过期好几年了。旧词根、旧映射、旧版本，没你想得那么值钱。”

“可你现在还在用。”秦荔冷笑，“连门都是拿它开的。”

音箱里安静了半秒，像对面的人笑了一下，只是笑意很薄。

“平台每天要给几万条东西找名字。谁先交一版，谁先占一格；哪一格能铺出去，要看系统；哪一版最后进档，要看留存结果。规则一直很清楚。”

“清楚个鬼。”秦荔盯着屏幕，手上已经在拷贝，“你们把别人的首稿拆开拿走，再把写回档案那只笔攥在自己手里。难怪所有事都越叫越像。”

脚步声越来越近，已经到了门外。

谢识雨的声音仍旧不紧不慢：“像，才好管理。太具体的名字挂不久，太私人也挂不久，只有能复用的句子最省成本。”

陆既明忽然开口：“所以你们开始偷私话。”

音箱那头顿了一下。

“陆既明。”谢识雨念他的名字时，语气还挺客气，“你比她更快看懂结构。可惜结构不归你。”

“今晚开始，未必。”

陆既明说完，俯身按下回滚模拟的最后一个按钮。他没去导出整张表，反而照着秦荔刚才那句童年注释，直接把一行字塞进了系统的“终稿回流申请”里。

`别人乱改标题，要把稿子退回来。`

秦荔瞬间反应过来，眼睛一亮。

这一下比截图狠得多。截图只是把证据带走，回流申请会逼系统自己往上找，到底谁有资格把名字退回给主人。

屏幕猛地一闪。

整个评审室的墙屏全亮了。热搜、标签、标题、推荐四个模块像被同一根线拽住，齐刷刷往上翻，翻得很快，最顶上终于露出一截原本藏起来的树形目录。

热搜之上是标签，标签之上是标题，标题之上是推荐，再往上还有一层细得像影子的分支。

`命名`

而 `命名` 上面，悬着一条灰掉的、更高的枝。

`写作`

秦荔呼吸一滞。

门把手已经被人从外面压下去了。陆既明一手拔掉 U 盘，一手扣住她手腕：“走。”

“再等一眼。”秦荔死盯着那条灰枝，“它下面还有东西。”

果然，`写作` 两个字底下缓缓滑出一条待执行项目，像一封刚露角的信。字还没完全展开，门已经撞开一条缝，保安的鞋尖先顶了进来。

陆既明几乎是半抱半拽，把秦荔从椅子里拉起来。他们擦着长桌往侧门冲，屏幕最后亮出的那几行字，从秦荔眼底狠狠划过去。

`上层分支：写作`
`访问级别：平台不可读`
`排队任务：样本对照组-2 / 私话校准`
`上线时间：05:00`

秦荔心口猛地一缩。

私话。

他们只在床边、车里、洗完澡靠着说过的那些话。

“跑！”唐照在耳机里快喊破音了。

两个人冲进安全通道，楼梯间的感应灯一层层炸亮。秦荔跑得急，鞋跟在台阶边上磕了一下，整个人往前一晃，陆既明伸手把她往自己这边一带，另一只手撑着扶手，几乎把她整个人拢在身前。楼上保安的脚步声已经追下来，楼道里全是回音，空气热得发硬。

“U 盘给我。”秦荔喘着气说。

“先下楼。”

“给我。”

陆既明把 U 盘塞进她掌心，低头看了她一眼，目光又黑又沉：“下去以后，你先上车。我回头拖他们十秒。”

“你敢。”秦荔一把拽住他衣领，声音又急又凶，“你今天敢把我一个人先丢车里，我明天就把你名字改成楼道逃兵。”

陆既明竟然笑了一下，胸口还在起伏，眼里那点火却全是偏着她的。

“行，一起。”

他们冲到地下二层时，林穗发来的车位定位刚好跳出来。黑色商务车停在消防门边，车灯没开，像一只趴着不动的兽。唐照把后门远程弹开，周遥那边终于发来一句短消息。

`合作方电话我替你们顶住了。东西带回来。`

秦荔钻进车里，车门“砰”地合上。直到这时，她绷了一整路的肩背才猛地塌下一点，手心全是汗，U 盘边角把掌纹都硌红了。陆既明紧跟着上车，反手把门锁死，呼吸还有些乱，额角全是薄汗。

车里很暗，只有仪表盘一层冷蓝。

秦荔低头看着那只 U 盘，忽然说：“他们在五点上线第二批。”

“我看见了。”

“还写了私话校准。”她抬眼，眼底那点火已经烧得发亮，“这帮人真敢动。”

“嗯。”

“陆既明。”

“我在。”

她什么也没再说，直接扑过去拽住他衣领，狠狠亲了上去。

这个吻来得又快又重，带着整晚憋出来的火气和后怕，撞得两个人呼吸都乱了。陆既明手掌一下扣住她后腰，把人按得更近，唇齿间全是急促发热的气息。秦荔在他下唇上咬了一口，很轻，又像故意留个印。下一秒她自己先退开一点，额头还抵着他，气息擦得发颤。

“这一下是回执。”她说。

陆既明盯着她，声音低得发哑：“收到。”

秦荔把 U 盘塞进他掌心，呼吸还没平：“回去先拆树。五点之前，我要知道他们偷了我们哪句私话。”

陆既明捏紧那只 U 盘，另一只手还压在她腰后没松：“好。”

车子发动，地下车库的灯带一格一格往后退。秦荔偏过头，看向屏幕上刚同步出来的截屏。那条灰色分支还挂在最上面，安静得像一把悬着的刀。

`写作`

它上面没有人名，没有说明，没有入口。只有底下那条新弹出来的排队任务，冷冰冰地亮着。

`样本对照组-2 / 私话校准 / 05:00`

而五点，还剩一小时四十六分。
