# 《全宇宙都在围观我和竹马热恋》写作风格指南

## 目标手感

- 读起来要像一口气滑过去：句子有流速，段落有镜头，情绪一层压一层
- 要甜，但不是糖水；甜感来自熟悉、偏爱、共同作战和高压场景下的身体靠近
- 要大，但不突兀；世界观每次升级都必须由上一层自然推出

## 单章节奏

- 开章 150 字内尽量进入现场
- 每章至少完成一件现实推进和一件关系推进
- 中段必须有一次由秦荔点燃、陆既明立刻跟上并把局面推大的节奏抬升
- 章尾必须留下下一章钩子，不能平收
- 术语必须后置，先让异常以动作、画面、损失和结果出现，再命名它

## 整书节奏

- 每 2 章必须出现一次不可逆转折
- 每 8 章必须出现一次层级升级：从项目到城市，从城市到平台，从平台到多城，从多城到宇宙写作权
- 亲密升级和甜蜜升级都不能慢于世界观升级：每 2 章至少新增一种亲密或甜蜜维度
- 前 8 章内必须落下第一次真正的高热关门戏或正面床戏，之后不允许热度和甜度停滞
- 前 16 章埋主伏笔，17-48 章滚雪球，49-64 章进入连续疯收
- 任何“过渡章”都必须被改写成推进章

## 甜感规则

- 两人始终站在同一边
- 可以斗嘴，但不冷暴力
- 可以嘴硬，但身体和行动不能犹豫
- 亲密互动要像长期情侣，不像暧昧期新人
- 不能只写公开护短，也要写同居感、赖床、喂食、抱着不撒手、床边余温
- 可以出现更明确的性张力、正面床戏和事后余温，但必须自然，不得为了“开车”硬拐弯
- 上床以后还要继续升级甜蜜和亲密，不准把“已经做过了”写成终点
- 如果写到控制感、命令感、轻度束缚或 SM，必须明确自愿、边界清楚并保留事后照料
- 甜感不是插播福利，而是改变世界规则的执行器
- 每卷至少要有一次高热关门戏、余温戏或明确带欲望的关键场面，但前 8 章内就必须先落下一次真正升温

## 元叙事规则

- 允许角色意识到“章节”“标题”“观众”“被阅读”“被命名”
- 元叙事出现后，必须立刻作用于事件、关系或规则
- 不允许用长段解释把小说写成设定手册
- 终局的大回收必须来自前文埋设，不得临时加设定硬解释

## 句法禁令

- 少用“你不是……你是……”和“不是X，而是Y”的正名句式，优先让人物直接下判断
- 少用“稳稳接住”“稳稳托住”“被世界接住”这类悬浮托举词，改写成可见动作、姿态、工具或后果
- 少用“被看见、被命名、被尊重”这类抽象三连，优先写具体代价、具体劳动和具体回声
- 少把“xx权”“xx债”“接口”“门”直接砸进正文，先写现场，再让人物把它说出来
- 少用硬拔高的比喻和莫名其妙的暗喻，网文正文优先清楚、利落、上头

## 创作前置动作

每次新章节创作前都要执行一次风格抽样：

```bash
npm run paimon:style-sample -- --source-path state/drafts/style-corpus/longform-reference.txt --label quanyuzhou-relian-chXX
```

要求：

- 随机抽取连续约 20000 字
- 生成风格摘要并保留审读记录
- 只模仿语言节奏、句法密度、叙述呼吸与镜头组织
- 严禁复用样本文本中的人物、设定、桥段与冲突
